Переход на другую лицензию

Если вы уже используете «Старт», «Стандарт» или другую редакцию продукта, вы можете расширить возможности вашего сайта - добавить новые возможности для ваших посетителей (создать блоги, форумы, фотогалереи, отправлять рассылки, проводить опросы), открыть интернет-магазин, управлять партнерскими сетями, анализировать статистику посещений, определять эффективность ваших рекламных кампаний и многое другое.

Для этого вам нужно перейти на новую редакцию продукта. При переходе вся информация на сайте сохранится. Вам не потребуется создавать сайт заново. С помощью технологии SiteUpdate вы получите новые модули продукта и установите их без помощи разработчиков.

Стоимость перехода на другую редакцию продукта составляет разницу в цене редакций.

Правила продления

Продление для редакции «Сайт 1С:Франчайзи» приравнивается к продлению для редакции «Стандарт», а для продления редакции «1С-Битрикс: Интернет-магазин 1С:Франчайзи» приобретайте продление для редакции «Бизнес».

Покупка продлений на два года не допускается.
